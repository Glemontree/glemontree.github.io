    • 概述

    link标签，当在文档中声明使用外接资源（比如CSS）时使用此标签，link是单独出现的

    属性

    • href – 指定需要加载的资源(CSS文件)的地址URI
    • media – 媒体类型
    • rel – 指定链接类型,设定是指对象和链接目标的关系,可选值, link还可以用Shortcut Icon等
    • rev – 指定链接类型
    • type – 指定所连接文档的MIME类型，css的MIME是type/css,一般使用type=”text/css”
    rel属性

    link标签中的rel属性，定义了文档和链接的关系，rel属性通常出现在a，link标签中，其通常具有下列属性值：

    • Alternate – 定义交替出现的链接
    • Stylesheet – 定义一个外部加载的样式表
    • Start – 通知搜索引擎，文档的开始
    • …
