    这样即使通过编译，链接器也会报错，因为只有声明而没有具体的实现。那么能不能直接把拷贝构造函数和赋值函数声明在public下，只是不实现？这样也是可以的，因为链接器不会放过这个只有空架子的声明而没有具体的实现，但这不是一种好的解决方法，因为编程的一个原则是：错误越早被发现越好，编译要早于链接，所以在编译阶段能发现的错误就不要拖到链接阶段。
