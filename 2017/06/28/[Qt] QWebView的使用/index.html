Qt中使用QWebView

在Qt中要使用QWebView必须经过一些配置，否则会出现error:C1083: 无法打开包括文件:”QWebView”这样的错误，这主要是因为QWebView、QWebFrame、QWebPage、QWebInspector等这些类被单独移到了QtWebKitWidgets模块，不再在QtWebKit模块当中。解决方法如下：

  • 在*.pro文件中添加Qt += webkitwidgets；
  • 在源文件中添加#include <QtWebKitWidgets/QWebView>。

VS中使用QWebView

我个人比较喜欢在VS中开发Qt程序，不太喜欢Qt的界面，那么在VS中怎么使用QWebView呢？需要进行下面的配置：

  • 在解决方案资源管理器中右键你的项目，选择Qt Project Settings；
  • 选择“Qt Modules”选项卡，勾选WebKit和WebKit Widgets选项即可。
